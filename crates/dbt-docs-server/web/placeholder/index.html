dbt docs — placeholder UI

The Rust HTTP server is running, but the React SPA hasn't been built. The API is fully functional; only the UI is missing.

Build the SPA

cd fs/sa/crates/dbt-docs-server/web
npm install   # requires GITHUB_TOKEN with read:packages for @dbt-labs/sourdough
npm run build

Then re-run cargo build -p dbt-cli — `rust-embed` picks up the freshly built bundle and the next dbt docs serve ships the real UI.
